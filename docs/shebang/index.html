Creating scripts using emuto

The shebang for emuto is #! emuto -s.

To create an emuto script simply create a file named my_script.emu, with the following that shebang in its first line. For example:

my_script.emu

#! emuto -s

{
  "static": {
    "data": 42
  }
}

Make your script executable:

chmod +x ./my_script.emu

Now you can run your script just like any other script:

./my_script.emu

{
  "static": {
    "data": 42
  }
}
